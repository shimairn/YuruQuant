# 04 运行手册

## 环境

- Python: `3.13.11`
- Conda 环境：`minner`
- GM SDK：安装在 `minner`

## 命令

```powershell
# 回测模式
conda run -n minner python main.py --mode BACKTEST --config config/strategy.yaml

# 实盘模式
conda run -n minner python main.py --mode LIVE --config config/strategy.yaml

# 烟测
conda run -n minner python -m unittest discover -s tests -v
```

## 运行行为

- 若未配置 `gm.token` 或 `gm.strategy_id`，自动使用本地仿真运行。
- 仿真运行会执行完整回调链并输出两类 CSV 报表。

## 输出

- `reports/trade_report.csv`
- `reports/daily_report.csv`
