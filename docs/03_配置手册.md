# 03 配置手册

配置文件：`config/strategy.yaml`

## 顶层结构

- `runtime`：运行模式、品种、周期、订阅窗口
- `strategy`：缠论与突破过滤参数
- `risk`：单笔风险与出场参数
- `portfolio`：组合层风险参数
- `gm`：GM 凭据与回测时间
- `reporting`：报表开关与输出路径

## 覆盖规则

1. CLI 参数（`--mode`）
2. YAML 配置
3. 环境变量（仅 GM 凭据）：`GM_TOKEN`、`GM_STRATEGY_ID`

## 默认参数（核心）

- `risk.risk_per_trade: 0.012`
- `risk.hard_stop_atr: 2.8`
- `risk.first_target_r_ratio: 2.0`
- `risk.trail_activate_r: 1.0`
- `risk.time_stop_bars: 12`
- `risk.max_stopouts_per_day_per_symbol: 2`
- `strategy.require_next_bar_confirm: true`
- `strategy.atr_pause_ratio: 2.0`
- `strategy.atr_pause_lookback: 50`

## 已移除参数（当前版本不再支持）

- `strategy.reentry_cooldown_bars_after_stop`
- `strategy.reentry_min_distance_atr`
- `strategy.platform_reset_required`
- `risk.cost_cover_mult`
